Package: com.stehno.vanilla.io

[Groovy] Class RollingFile

    • com.stehno.vanilla.io.RollingFile

    • Decorator used to provide rolling file behavior for a standard File object. The rollover triggering is based on a strategy provided by the configured instance of the RolloverTriggerStrategy interface. The resulting rollover file names (and possibly locations) are provided by the configured instance of the RolloverFileProvider instance. By default, if not otherwise specified, the FileSizeRolloverTriggerStrategy and DefaultRolloverFileProvider are used. The rollover files may also be compressed. The rollover behavior is thread safe.
