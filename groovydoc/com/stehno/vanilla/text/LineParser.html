    • Method Detail

      • boolean parsable(java.lang.String line)

        Determines whether or not the provided line is parsable. A non-parsable line is not necessarily invalid, it may simply be a comment or otherwise ignored line, dependent on the implementation.

        Returns:
        true if the line is parsable and should be parsed
        Parameters:
        line - the line being parsed

      • java.lang.Object parseItem(java.lang.Object item, int index)

        Parses an individual item from the line being parsed. This allows for specific types of objects to be created when parsing a line, rather than just Strings.

        Returns:
        the parsed item which should be a String by default (if not otherwise processed)
        Parameters:
        item - the chunk of the line containing a single line item
        index - the chunk index (column index)

      • java.lang.Object[] parseLine(java.lang.String line)

        Parses the line of the file into an array of Objects.

        Returns:
        the array of Objects representing the parsed line data.
        Parameters:
        line - the line of the file being parsed
