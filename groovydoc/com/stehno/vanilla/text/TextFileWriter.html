Package: com.stehno.vanilla.text

[Groovy] Class TextFileWriter

    • com.stehno.vanilla.text.TextFileWriter

    • Utility component for writing object data to a file as one line of text per object written. A configured LineFormatter is used to write the text to the file. Comments may also be written to the file and these will be passed through the LineFormatter so that they may be appropriately formatter for the implementation. This component will not be very performant. It is intended for non-critical path operations.
