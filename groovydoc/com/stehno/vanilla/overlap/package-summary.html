Package com.stehno.vanilla.overlap

Package com.stehno.vanilla.overlap Description

The overlap API stems from a demo library used in my blog posting about how to determine the overlap of objects based on multiple variable axes. See Determining Overlap Across Multiple Variables
  • Interface Summary 
    Interface Description
    Lobe Lobes are the discrete dimensions of an overlap comparison, e.g. a single variable of an object used to build the whole overlap-comparison basis.
  • Trait Summary 
    Trait Description
    Overlappable Trait providing a means of determining whether or not two objects overlap.
  • Class Summary 
    Class Description
    ComparableLobe A ComparableLobe is a Lobe implementation providing support for single Comparable values, Comparable range values and combinations of either type.
    OverlapBuilder The OverlapBuilder is used to build the overlap comparison by appending the lobes for comparison.
