== Introduction

The Vanilla library is a multi-purpose utility library containing a lot of "what if" and experimentation code that actually seemed to have a
potential use, so it was made publicly available.

The project web site is: http://stehno.com/vanilla

The source code for the project is hosted on GitHub at: https://github.com/cjstehno/vanilla

Vanilla is licensed under the http://www.apache.org/licenses/LICENSE-2.0[Apache 2] open source license.
