Package: com.stehno.vanilla.jdbc.mapper

[Groovy] Interface ResultSetMapper

  • Defines a mapper used to map fields of a ResultSet into Java or Groovy objects.

    • Method Detail

      • java.lang.Object call(java.sql.ResultSet rs)

        Executes the mapper on the current row of the provided ResultSet.

        Returns:
        the mapped data extracted from the ResultSet row
        Parameters:
        rs - the ResultSet being operated on

      • java.lang.Object mapRow(java.sql.ResultSet rs, int row)

        An alias to the call(ResultSet) method used to facilitate simple inter-operation with the SpringFramework RowMapper interface. A ResultSetMapper should be usable as a RowMapper by simply casting it (in Groovy). 

        
     RowMapper rowMapper = myResultSetMapper as RowMapper
        Returns:
        the object value of the mapped row
        Parameters:
        rs - the ResultSet being operated on
        row - the current row (not used)
