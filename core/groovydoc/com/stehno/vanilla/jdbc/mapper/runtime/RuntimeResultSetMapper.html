[Groovy] Class RuntimeResultSetMapper

    • com.stehno.vanilla.jdbc.mapper.runtime.RuntimeResultSetMapper
    • All Implemented Interfaces and Traits:
      ResultSetMapper

      @groovy.transform.TypeChecked
class RuntimeResultSetMapper
extends java.lang.Object

      Dynamic runtime implementation of the ResultSetMapper interface. The mapping information is immutable and the actual mapping and extraction operations occur at runtime. The mapper is reusable and thread-safe.
