      ObjectMatcherDsl

      class ObjectMatcher
extends java.lang.Object

      An Object Matcher is a utility for determining whether or not an Object of a specified type matches a set of criteria configured via DSL. A matcher could implement actual equivalence, or it could ensure that an object is within a specified set of tolerance levels. An object matcher is defined using a simple DSL: 

           ObjectMatcher matcher = ObjectMatcher.matcher(Person){
         typeMatcher String, { v-> v != null }
         propertyMatchers([
             age: { v -> v > 18 && v < 40 },
             employed: { v-> v }
         ])
     }

     boolean matches = matcher.matches(somePerson)
     float score = matcher.score(somePerson)
      All properties and property types defined in the configuration will be used as the basis of match comparison, all others will be ignored.
